サマリー

クオンツ・金融分野を志望しています。数理的な素養と金融への強い関心を軸に、データ分析とAIエージェント活用を組み合わせて成果につなげることを得意としています。

  • AI・機械学習開発力：将棋の大規模ResUNet・板情報のTLOB（Transformer）・ルービックキューブ手順のLightGBM（勾配ブースティング）など、異なるドメイン・手法でモデルの学習を回し成果につなげてきました。タスクに応じて適切なアーキテクチャを選び、学習を完遂できる点が強みです。
  • 数理・分析力：数学検定準1級。R / Python で時系列データの前処理・回帰分析を行います。
  • 金融知識：日本証券アナリスト協会CMAに挑戦中、簿記2級取得。四季報・日経新聞を日課とし、市場と経済への関心が高いです。
  • AIエージェント活用：Claude Code / Codex を駆使し、企画から実装・公開まで一貫して短期間で複数のWebアプリを制作しています。
  • 再現性のある成果：目隠しルービックキューブでアジア2位・日本記録更新（3回）。一つの目標を頂点まで突き詰めた経験があります。

AI・機械学習開発

異なるドメイン・手法でモデルの学習を回し、成果につなげてきました。タスクに応じて適切なアーキテクチャを選び、学習を完遂できる点が強みです。

  • 将棋AI：PyTorchによる大規模ResUNetの学習（コンピュータ将棋選手権に出場し、二次予選まで進出）
  • 板情報：日本株の板情報を対象に、データ前処理・TLOB（Transformer for Limit Order Book）による学習・バックテストまで一貫して実施
  • ルービックキューブ手順：LightGBM（勾配ブースティング）で手順の回しやすさを学習・点数化（下記「制作Webアプリ／human-friendly-sorter」参照）
